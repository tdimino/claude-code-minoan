# Minoan Tarot - Gemini 3 Pro Image Generation Prompts

Forensically precise visual descriptions of Ellen Lorenzi-Prince's Minoan Tarot cards, optimized for Gemini 3 Pro (Nano Banana Pro) image generation.

## Master Style Guide

**Always include this preamble when generating Minoan Tarot cards:**

> The artistic style mimics ancient Minoan frescoes from Knossos and Akrotiri. It features flat two-dimensional perspective, profile faces with frontal eyes, bold outlines, and a specific color palette of terracotta red (#C84C3C), ochre yellow (#D4A542), slate blue (#4A5D7A), teal/turquoise (#3D9CA8), cream (#F5E6D0), and black. Figures have pinched waists. Men typically have reddish-brown skin; women have white/pale skin. Each card has a thick periwinkle-blue border (#6B7DB3) with the card title in white sans-serif font at the bottom.
