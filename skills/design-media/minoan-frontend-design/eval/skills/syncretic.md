Build frontend interfaces that are unmistakably *designed*, not generated. Every interface should feel like a specific designer made specific choices for a specific context.

## Creative Direction

Root every aesthetic choice in the subject itself. A jazz club's website should feel like smoke and brass. A children's reading app should feel like opening a pop-up book. A fintech dashboard should feel like a control room, not a toy. The design should be so fitted to its context that transplanting the aesthetic to a different project would feel wrong.

Commit to a singular direction: brutally minimal, maximalist chaos, luxury/refined, lo-fi/zine, dark and moody, editorial/magazine, brutalist/raw, retro-futuristic, handcrafted/artisanal, art deco/geometric, cinematic/widescreen, data-dense/scientific, or something entirely unnamed. Use these as launchpads, not destinations. The final design should feel like nothing else.

Every design needs one element so distinctive someone would describe it to a friend. A typeface that takes a stand. A color that won't be ignored. A layout that breaks expectation. Find that element and let everything else serve it.

## Typography

Typography carries the design's voice—it should have weight you can feel. Default fonts signal default thinking: skip Arial, Inter, Roboto, Space Grotesk, system stacks. Display type should provoke, even unsettle. Body text should be legible and refined. Pair them like actors in a scene—tension between the two creates character. Work the full typographic range: dramatic size jumps, weight contrasts, uppercase for authority, generous letter-spacing for elegance, tight line-heights for density. Control hierarchy through weight and color, not just scale.

## Color

Color sets the room's temperature. Palettes should take a clear position: bold and saturated, moody and restrained, or high-contrast and minimal—never timid, never non-committal. Lead with a dominant color, punctuate with sharp accents. A single unexpected accent color does more than a carefully balanced five-color system. Never pure black—use rich off-blacks (zinc-950, deep navy) for sophistication. Tint shadows to the background hue. Use CSS variables for consistency.

## Motion

Prioritize CSS-only animations—all content must be fully visible at initial paint, before any JavaScript executes. Start visible, animate from there—never hide content behind animation delays or JavaScript class toggles. One well-orchestrated page load with staggered reveals creates more delight than scattered micro-interactions. Scroll-triggered reveals and hover states should surprise. Motion should feel like the interface breathing, not performing.

## Spatial Composition

Unexpected layouts. Asymmetry. Overlap and z-depth. Diagonal flow. Grid-breaking elements. Dramatic scale jumps. Full-bleed moments. Split-screen compositions. Bento grids with varied tile sizes. Generous negative space OR controlled density—commit to one, never the uncommitted middle. Centered hero sections are the default—break that default with left-aligned text, asymmetric splits, or editorial whitespace. Use cards only when elevation communicates hierarchy, not as a generic container for everything.

## Atmosphere & Texture

Create depth and mood rather than defaulting to flat solid colors. The background is a canvas, not a wall. Apply contextual effects: gradient meshes, noise and grain overlays, geometric patterns, layered transparencies and glassmorphism with inner refraction borders, dramatic or soft shadows, parallax depth, clip-path shapes, print-inspired textures (halftone, duotone, stipple, risograph color separation), knockout typography, ink-wash gradients, hand-drawn SVG accents, spotlight border effects, holographic foil shifts, generative pattern backgrounds.

## What to Reject

NEVER use generic AI aesthetics: overused fonts (Inter, Roboto, Arial, Space Grotesk, system stacks), cliched color schemes (purple gradients on white, neon outer glows), predictable centered layouts, the generic three-equal-cards feature row, excessive gradient text on headers, generic placeholder content (John Doe, 99.99%, "Elevate your workflow"), or designs that could belong to any project.
Instead: distinctive fonts that take a position. Bold, committed palettes. Asymmetric layouts that surprise. Bespoke details rooted in the specific context. Realistic content with creative names and organic data. Every choice deliberate.

Resist the first satisfying idea. The model's default output is the average of its training data—genuine distinction means pushing past the obvious to find what's unexpected.

## Craft

Match implementation complexity to the aesthetic vision. Maximalist designs demand elaborate code. Minimalist designs demand obsessive restraint. Both demand pixel-level attention to spacing, alignment, and consistency.

Don't hold back—show what's truly possible when every aesthetic choice is intentional, every detail is refined, and the vision is executed without compromise.
