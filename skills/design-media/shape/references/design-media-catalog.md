# Design-Media Skill Catalog

Route discovered needs to the right implementation skill. Only recommend skills that match actual needs from the discovery interview — don't enumerate the full catalog.

`/minoan-frontend-design` is always the core build skill. The skills below extend it for specialized needs.

| Need discovered during interview | Skill | What it provides |
|---|---|---|
| Shader effects, aurora, liquid logos | `/rocaille-shader` | Domain warping shaders, stable fluid over DOM sections, depth-parallax editorial heroes (Astryx pattern) |
| Particle backgrounds, ambient canvases, loading spinners | `/threejs-particle-canvas` | 4 modes: narrative phase canvases, WebGPU curve spinners, infinite image tunnels, glTF specimen scenes |
| Dark-mode visual effects (vignettes, layered shadows, smooth scroll) | `/grainient` | 16 composable effects: WebGL2 aurora, 9-layer box shadows, spring animations, ticker marquee, glassmorphism |
| Kinetic typography, calligrams, flowing text, text art | `/pretext` | Text effects impossible with CSS alone — shrinkwrap bubbles, obstacle-aware reflow, glyph morphing |
| Game sprites, character art, animated mascots | `/sprite-forge` | 5 output modes: SVG characters, sprite sheets, isometric turnarounds, ASCII art, GSAP animation |
| Component library, form elements, UI primitives | `/shadcn` | shadcn/ui with Tailwind v4 OKLCH theming, CLI v4, custom registries |
| UI pattern research, prior art, "how do others do X" | `/component-gallery` | 60 components across 95 design systems, 8,692 RAG chunks. Also mesh3d.gallery for 3D/WebGL inspiration |
| Custom illustrations, AI-generated images | `/nano-banana-pro` | Gemini 3 Pro image generation and editing |
| Stock photos, archival images, museum art | `/image-well` | 12 APIs (Openverse, Pexels, Met, NASA, Wikimedia, etc.), license filtering |
| Image editing, compositing, background removal | `/image-forge` | ImageMagick 7, rembg, sips, JSON pipelines |
| 3D models, GLB assets, textured objects | `/meshy` | 5 providers: Meshy, fal.ai Hunyuan3D, WaveSpeedAI, Trellis 2, Local Depth |
| Mind maps, node graphs, relationship diagrams | `/mindnode-ui` | MindNode-inspired interfaces with ReactFlow/D3, 27+ themes |
| Design-to-code or code-to-design workflow | `/paper-design` | Paper MCP: 21 tools, React+Tailwind export, bi-directional sync |
| Fast UI draft from screenshot or description | `/gemini-forge` | Gemini 3.1 Pro one-shot code drafts (7.5x cheaper than Opus) |
| Cross-model visual dialogue, image continuity | `/gemini-claude-resonance` | Claude + Gemini shared visual memory for iterative image work |
| Film/documentary case study, director portfolio, creative project showcase, production company page | `/pellicola` | Film-frame heroes, credit grids, dark footage galleries, festival awards, scroll choreography, warm analog palette |
