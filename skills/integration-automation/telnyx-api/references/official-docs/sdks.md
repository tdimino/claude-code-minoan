Source: https://developers.telnyx.com/development/server-side-sdks

## Node.js (v2 — TypeScript, ESM)

The official Telnyx Node.js library for server-side applications. **SDK v2** (released Feb 2025) includes full TypeScript support and ESM module exports.

- [Documentation](https://developers.telnyx.com/development/sdk/node)
- Install: `npm install telnyx`
